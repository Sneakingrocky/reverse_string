reverse_string

This exercise was created to understand the work that Ruby's reverse method does work us.

<h1>What does Ruby's reverse method do?</h1>

<p>reverse → new_ary
Returns a new array containing self‘s elements in reverse order.</p>

<h1>Example 1</h1>

<code>
[ "a", "b", "c" ].reverse   #=> ["c", "b", "a"]
[ 1 ].reverse               #=> [1]</code>
